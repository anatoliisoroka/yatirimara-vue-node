# Invest-API - Node-Express-Graphql-Mongo-Firebase

## Overview
API for mobile application for therapist and patients.

API for admin panel.

## Tech stack

- node: v16.5.0
- mongodb: 4.4
- apollo-graphql
- firebase

## Deployment
### How to run
```bash
npm i
pm2 start src/server.js --name api
```

### Manual Deployment
```
bash /home/ubuntu/_api.sh
```
